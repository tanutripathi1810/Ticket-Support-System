# support desk system

This is a mern stack support ticketing system

### Tech stack

- MongoDB
- ReactJS
- NodeJS-Express
- Redux-Toolkit
- React-Router

### How do I get set up?

```
git clone <url>

npm install
```

### Contribution guidelines

- add `.env` file in root and add the following

```
NODE_ENV=development || production

PORT=5000

MONGO_URI=your mongo uri

JWT_SECRET=your jwt secret
```
